# C++ Calculator with Arithmetic Progression

## Description
This project is a menu-driven calculator built in C++.

It can:
- Perform basic arithmetic operations
- Calculate Arithmetic Progression (AP):
  - nth term
  - sum of n terms

## Features
- User-friendly menu
- Modular functions
- Handles division by zero

## Tech Used
- C++

## How to Run
```bash
g++ main.cpp -o calc
./calc
